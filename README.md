# IELTS 听写练习

## 启动

日常使用时，直接双击 `启动听写.cmd`。它会启动本地音频服务并打开浏览器。

在 `dictation` 目录运行专用本地服务：

```powershell
python server.py
```

然后打开 `http://localhost:4173`。全部英音存放在 `audio` 目录，网页播放时不再需要联网。现在也可以直接双击 `index.html` 使用。

学习数据保存在当前浏览器的 Local Storage 中。新词答对一次即标记为掌握；答错的词进入错题库，10 分钟后首次复习，首次复习答对后约 20 小时再次复习，连续答对两次后毕业。复习时再次答错会重置连续正确次数。

学习进度同时保存到 `data/dictation.db`。浏览器存储用于离线缓冲；SQLite 是长期记录。服务器部署方法见 `DEPLOYMENT.md`。

词库按打乱后的顺序每 30 个新词划分为一组。每组练习会混入所有已经到复习时间的错题；完成后可以进入下一组，也可以从顶部选择任意组。左上角保留上一题结果，直到提交下一题后更新。

顶部练习模式提供：

- `正常学习`：当前组 30 个新词与到期错题混合，继续使用原记忆曲线。
- `曾经错过`：练习所有历史上答错过的词，包括已经掌握并退出正常错题库的词。
- `自定义练习`：按组范围、学习状态和题数随机抽题。

额外练习中再次答错会重新进入正常错题复习；尚未到期的错题答对不会提前推进复习阶段。

## 错题重点等级

- 累计错 1 次：普通历史错题。
- 累计错 2 次：1 星重点。
- 累计错 3 次：3 星重点。
- 累计错 4 次及以上：5 星重点。

重点筛选采用包含关系：1 星包含所有 3/5 星，3 星包含所有 5 星。星级依据永久保存的累计错误次数计算，因此单词从正常错题复习毕业后，仍会保留在“曾经错过”和对应星级的自定义练习中。

提交答案后，左上角显示词库中的标准拼写，正确位置标绿、错误位置标红，并保持词库原有大小写。点击该标准答案可再次播放上一题的正确英音。

左上角上一题答案旁的 `?` 按钮用于反馈词条问题。反馈会保存到 SQLite 的 `feedback_log` 表，并同步追加到 `data/feedback-log.jsonl`。可以先预览待处理反馈：

```powershell
python apply_feedback.py
```

确认后应用修改：

```powershell
python apply_feedback.py --apply
```

脚本会根据反馈中的拼写、词义和发音朗读文本更新词库或重新下载对应本地音频。

## 发音来源审计

本地音频优先使用 Oxford Learner's Dictionaries 和 Cambridge Dictionary 页面中的 UK/英式真人词典音频。批量审计脚本：

```powershell
python audit_official_audio.py --apply
```

审计结果写入 `official-audio-report.json`，未匹配到官方词典真人音频、继续保留 Google `en-GB` TTS 的词条列在 `official-audio-fallbacks.md`。

错题库每一行都可以单独删除。删除后该词会永久退出当前错题、曾经错过、1/3/5 星重点和自定义练习，不再参与记忆曲线复习；排除状态保存在 SQLite 中并随服务器同步。
